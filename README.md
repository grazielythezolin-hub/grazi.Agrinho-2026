# grazi.Agrinho-2026
Projeto Agrinho 2026
Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”. 

Link do site:https://grazielythezolin-hub.github.io/grazi.Agrinho-2026/

Tema: Agricultura Orgânica: Produção de Alimentos sem Agrotóxicos e seus Benefícios Ambientais. 
 
Objetivo: Identificar as vantagens de uma dieta livre de defensivos químicos, Demonstrar como as técnicas orgânicas evitam a erosão e mantêm a terra fértil, Investigar o crescimento das vendas e o perfil desse comércio.
